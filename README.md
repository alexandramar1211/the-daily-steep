# The Daily Steep

> Ten minutes, just for you. Music · Tarot · Tea.

A daily ritual app that pairs your mood and playlist with a tarot card reading and tea recommendation.

## Deploying

This is a static site. The entire app lives in `public/index.html`.

### Vercel (recommended)
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → New Project → Import your repo
3. Set **Output Directory** to `public`
4. Deploy ✓

### Railway
1. Push this repo to GitHub
2. Go to [railway.app](https://railway.app) → New Project → Deploy from GitHub
3. Add a **Static Site** service, set root to `public`
4. Deploy ✓

## Custom Domain
Point your Squarespace domain to Vercel by adding the DNS records Vercel provides after deployment.
